# CAR-POOL
College carpool system nmims

Beautiful and pleasant NMIMS Chandigarh community carpool website made with **HTML, CSS, and JavaScript**.

## Features
- Student and teacher registration/login.
- Only `@gmail.com` and `@nmims.in` emails allowed.
- Pickup area restricted to Chandigarh, Panchkula, Zirakpur.
- Drop location fixed to NMIMS Chandigarh.
- Every logged-in user has both rider and requester sections.
- Rider can create rides, set seats, and use temporary off/on.
- Requester can request rides.
- When seats are full, ride status becomes `full` automatically.
- No payment flow (community college carpool).

## Run
Open `index.html` in browser.

> Data is stored in localStorage for demo usage.
